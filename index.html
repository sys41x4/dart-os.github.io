Welcome to Dart OS

Dart OS is a Linux-based operating system designed for offensive and defensive security, and other security-related tasks.
Currently it is in alpha stage, and is continously developed and contributed by Arijit Bhowmick [sys41x4]

Features

Powerful command-line interface

Dart OS includes a powerful command-line interface that allows you to easily perform complex security-related tasks.

Wide range of security tools pre-installed

Dart OS comes with a wide range of pre-installed security tools, so you don't have to waste time installing them yourself.

Customizable interface

Dart OS allows you to customize the interface to suit your needs, making it easy to use and navigate.

[A] Add dartos-experimental Repository to APT Sources

Currently dartos-experimental branch is accessable to use before a beta version of this OS is pushed

Add dartos-experimental branch in your repository as : 

sudo sh -c 'echo "deb [arch=amd64] http://dart-os.github.io/deb/dartos dartos-experimental main contrib non-free non-free-firmware" > /etc/apt/sources.list.d/deb.dart-os.list'


Add dartos-experimental branch's GPG Key as:
curl https://dart-os.github.io/keys/D0626CA83E6C820B/ | grep -o '[^[:space:]]*\.gpg' | cut -d'>' -f 2 | xargs -I {} sudo wget https://dart-os.github.io/keys/D0626CA83E6C820B/{} -O /etc/apt/trusted.gpg.d/{}
